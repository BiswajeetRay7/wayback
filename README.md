## About
Wayback is a tool for crawling urls for targets from wayback implementation in bash.

## Features

- Easy to Install
- Easy to Use
- Designed In Bash
- Easy To Execute 

## How to Install

### Prerequisites

- Git 

### Installation

- Clone the Repo
  ```bash
  git clone \///////////////////////
  ```
- Open cloned directory
  ```bash
  cd wayback && chmod +x wayback.sh 
  ```
- Now run tool
  ```bash
  npm install 
  ```
> Usage:
node . 
OR
npm start
